Projects

[Project #1] CCNA Project: Network Infrastructure Simulation

This project involved designing and implementing a robust, secure, and redundant network infrastructure for a simulated multi-floor branch office using Cisco Packet Tracer. It showcased advanced routing, switching, and security configurations.

  • Hierarchical Network Design: Designed and implemented a complex network structure featuring switches, dual-core routers, and ISP connectivity, establishing a robust and scalable foundation.
  • Secure Access & VLANs: Configured secure remote access via SSH and implemented essential security measures including Port Security and native VLANs to protect network integrity.
  • Dynamic Routing & Redundancy: Implemented dynamic routing using OSPF and ensured network redundancy through HSRP across multiple VLANs, optimizing traffic flow and fault tolerance.
  • Critical Network Services: Deployed essential network services such as DHCP, DNS, PAT, Syslog, and SNMP to support seamless network operation and management.
  • Granular Traffic Control (ACLs): Created and applied Access Control Lists (ACLs) for granular traffic management, ensuring precise control over network access and data flow.
